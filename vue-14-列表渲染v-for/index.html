例子

v-for

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://cdn.jsdelivr.net/npm/vue@2.5.16/dist/vue.js"></script>
    <title>Vfor</title>
</head>
<body>
    <div id="app">
    <ol>
        <li v-for="item in items" :key="item.title">
            {{ item.title }}
        </li>
    </ol>
    <h1>Authors Info</h1>
    <ul>
        <li v-for="value,key,index in objs">
            {{ index }} : {{ key }} : {{ value }}
        </li>
    </ul>

    <h1>v-if in v-for</h1>
    <ol>
        <li v-for="todo in todos" v-if="todo.status==1">
            {{ todo.name }}
        </li>
    </ol>
    </div>
    <script>
        app = new Vue({
            el: '#app',
            data:{
                "items":[
                    {title: 'phyger'},
                    {title: 'fly'}
                ],
                "objs": {
                    age: 18,
                    name: "phyger",
                    home: "xian"

                },
                todos: [
                    {name: 'eat',status: 1},
                    {name: 'work',status: 1},
                    {name: 'sleep',status: 0}
                ]
                }
        })

    </script>
</body>
</html>

v-if in v-for 场景中，v-for 的优先级要高于 v-if，所以 v-if 会分别在每个 for 循环中重复运行。

效果

只显示 status=1 的任务。
